# React_Hook_Form
Formulário com React Hook Form

## Techs

* React
* Styled Components
* React Hook Form
* Yup 


## Starting Dev Environment

    * Run npm install or yarn install.
    * Run yarn start.
    * npm i styled-components
    * npm i react-hook-form
    * npm i yup

## Projeto

Um Formulário básico utilizando o React Hook Form
